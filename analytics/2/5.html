On-Chain Analysis Strategy

On-chain analysis studies blockchain data to understand Bitcoin's fundamental value drivers. The number of unique addresses reflects network adoption — more active addresses means more users, which supports price. The transaction count measures actual network usage. Signals are generated using Bollinger-style bands (30-day MA ± 2σ) on both metrics: a buy signal fires only when both addresses AND transactions break above their upper band simultaneously — indicating an abnormally strong network surge. A sell signal fires when both drop below their lower band — an abnormally weak period. This approach filters out noise and only flags truly exceptional on-chain movements, producing fewer but more reliable signals.
