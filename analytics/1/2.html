Ichimoku Kinko Hyo Strategy

Ichimoku is a complete trading system that provides support/resistance levels, trend direction, and momentum signals in a single glance. It consists of five lines: the Tenkan-sen (conversion, 9-period) and Kijun-sen (base, 26-period) act like fast and slow moving averages — a Tenkan cross above Kijun is a buy signal. The Cloud (Kumo), formed by Senkou Span A and B, defines the trend: price above a green cloud is bullish, below a red cloud is bearish. The Chikou Span (lagging line) confirms signals when it crosses above price. The strongest trades occur when all five elements align — this is called a "triple confirmation" and it drastically reduces false signals.
